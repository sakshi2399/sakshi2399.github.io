# Sakshi Portfolio (React + Vite)

## Getting started

1. Install dependencies
	- `npm install`
2. Run locally
	- `npm run dev`
3. Build for production
	- `npm run build`
4. Preview the production build
	- `npm run preview`

## Tech
- Vite + React 18
- Tailwind CSS with forms and typography plugins
- lucide-react icons

## Deploy
The build output lives in `dist/`. Serve it with any static host (Netlify, Vercel, GitHub Pages, S3). For GitHub Pages, run `npm run build` and publish the `dist` directory.
